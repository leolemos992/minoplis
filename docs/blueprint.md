# **App Name**: Localopoly

## Core Features:

- Multiplayer Gameplay: Allow multiple players to join a game session and interact in real-time.
- Board Creation: Enable users to create custom boards with different locations and properties.
- Character Selection: Let users choose from a variety of unique characters with different abilities and attributes. Use AI tool to select best character based on playing style.
- Property Management: Enable players to buy, sell, and upgrade properties to generate income.
- Event Handling: Implement random events and chance cards to add variety and unpredictability to the gameplay.
- Trading System: Allow players to trade properties and assets with each other.
- Victory Condition: Establish a clear win condition, such as bankrupting all other players or reaching a certain net worth.

## Style Guidelines:

- Primary color: Vivid blue (#3498DB) to represent the sky and open possibilities.
- Background color: Light gray (#ECF0F1), providing a clean and neutral backdrop for the game board.
- Accent color: Yellow-gold (#F1C40F) to highlight important elements like currency, properties, and interactive components.
- Font: 'PT Sans', a humanist sans-serif for both headings and body text, ensuring readability and a touch of warmth.
- Use simple, recognizable icons to represent properties, events, and actions.
- A clean, card-based design that gives each object on the board affordance
- Subtle animations to indicate when the user should interact with the UI
